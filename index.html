Welcome to Exoplanet Search! This is a website which allows you to query a large dataset of Exoplanets with specific properties, resulting in detailed, personalized statistics!


Not only this, but the website will also generate a planet map using the results of your query to provide data visualization. A more general analysis of the data can be found on the Insights page.

Key Features

Data Analysis

An investigation of Exoplanet characteristics which may influence chances of discovery.

Query Planets

Select a subset of planets from the Caltech Exoplanet Archive & generate personalized statisitcs.

Planet Map

Generate a planet map from selected planets showing their position relative to Earth.
